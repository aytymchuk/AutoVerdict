# AutoVerdikt Architecture Design

## Overview
High-level design and global scaffolding document for the AutoVerdikt platform.

## Architecture Modules
- `frontend/`: Single Page Application or Server-Side Rendered Web UI.
- `backend/`: Core API, services, and business logic.
- `landing/`: Static or lightly dynamic marketing site.
- `ascode/`: Infrastructure as Code (IaC) templates and deployment manifests.

*Note: Implementation code will be added in subsequent feature prompts.*
